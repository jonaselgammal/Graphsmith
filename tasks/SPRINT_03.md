# Sprint 03 — Local Registry and Search

## Goal
Publish and search skills locally.

## Deliverables
- local registry layout
- publish command
- search command
- metadata index
- filter search
- optional FastAPI service

## CLI
- `graphsmith publish <path>`
- `graphsmith search <query>`

## Search fields
- id
- name
- description
- tags
- effects
- input names
- output names

## Definition of done
A user can publish example skills and search them by text and filters.
